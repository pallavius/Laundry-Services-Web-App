🧺 Laundry Service Web Application

A responsive web application for a laundry service that allows users to explore services, add items to the cart, book appointments, and receive order confirmation emails using EmailJS.

This project was built as part of my MERN Stack learning journey to practice frontend development and dynamic user interactions.

🚀 Project Overview

The Laundry Service Web Application provides a simple and user-friendly interface where customers can:

Explore available laundry services

Add or remove services from their cart

View total pricing dynamically

Book a service appointment

Receive an order confirmation email using EmailJS

The application focuses on creating a smooth user experience with clean UI and responsive design.
